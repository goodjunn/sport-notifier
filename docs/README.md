# 每日運動通知小工具

公司內部使用的輕量通知工具，方便每天快速產生運動通知訊息，複製後貼到 Slack 即可。

---

## 為什麼有這個工具

為了讓有運動習慣的同事可以互相招呼、一起動起來，但又不想另外開頻道打擾其他人，所以建立這個小工具搭配 Slack 的固定 Thread 使用，把每日通知集中在同一串，不洗版、不打擾，有興趣的人自己訂閱那個 Thread 就好。

---

## 使用方式

1. 選擇今天的**運動類型**（律動／核心／燃脂）
2. 設定**開始時間**，視需要加上附加說明
3. 選擇或自訂**結尾號召語**
4. 點「複製訊息」，貼到 Slack 指定 Thread 即可

---

## 運動類型說明

| 類型 | 強度 | 代表項目 | 適合時機 |
|---|---|---|---|
| 🎵 律動運動 | 中 | Zumba、有氧舞蹈、韓舞、K-pop | 想跟著音樂動、輕鬆有趣 |
| 💪 核心運動 | 低～中 | Plank、橋式、捲腹、瑜珈流 | 飯後、狀態不佳、不想流太多汗 |
| 🔥 燃脂運動 | 高 | HIIT、Tabata、Combat、Burn & Tone | 想大量消耗熱量、挑戰自己 |

> 分類判斷小技巧：做完不太喘 → 核心／跟著音樂搖擺 → 律動／做完想躺下 → 燃脂

---

## Slack 使用建議

- 在指定頻道發一則**固定母訊息**作為每日通知區（例如：`【 不動就角落生鏽 🎵】`）
- 每次通知回覆在同一個 Thread，並勾選「同時顯示在頻道」
- 有興趣參加的同事只需追蹤該 Thread，不影響其他人
